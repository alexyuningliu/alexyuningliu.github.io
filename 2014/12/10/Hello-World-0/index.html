Legos and Code

“A master Lego builder!” At age 8, if you had asked me what I wanted to be when I grew up, I probably would have said something along those lines.

I absolutely loved playing with Legos - there was just something so immensely satisfying about taking simple little plastic blocks and building some thing amazing - maybe a flying pirate ship today, an indestructible fortress tomorrow, and jungle obstacle course over the weekend.

I guess, when it comes down to it, I’m learning to code because I want to recapture some of that creative energy and freedom I felt when playing with Legos when I was younger.

As I am exposed more and more to Javascript functions and objects, I realize that they’re not that different from Lego blocks - little reusable pieces of stuff that can be used to build even more amazing stuff when put together in the right ways. The whole turns out greater than the sum of its parts.

I’m also doing it because it’s a challenge. I know it’s not going to be easy - setting up this Hexo blog has already been an adventure on its own. But, when it finally clicks (just like when you finally finish that flying pirate ship sitting atop an indestructible fortress oh boy), it’s remarkable.

So here’s to Legos and code!
